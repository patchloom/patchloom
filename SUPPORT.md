- **Bug reports**: Open an issue using the
  [bug report template](https://github.com/patchloom/patchloom/issues/new?template=bug-report.yml).
- **Feature requests**: Open an issue using the
  [feature request template](https://github.com/patchloom/patchloom/issues/new?template=feature-request.yml).

  [GitHub Discussions](https://github.com/patchloom/patchloom/discussions).
